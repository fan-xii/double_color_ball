# 双色球模拟器

完全遵循中国福利彩票双色球机制与开奖规则的模拟器，基于 React + Vite 构建。

## 功能特性

- **手动选号**：红球 33 选 6，蓝球 16 选 1，完全遵循双色球规则
- **随机选号**：一键随机生成号码
- **单注 / 批量模拟**：支持 1 注 ~ 1 亿注批量开奖，实时反馈中奖结果
- **多线程并行**：利用 Web Worker 多线程并行计算，1 亿注也能流畅运行不卡顿
- **进度条显示**：大批量模拟时实时显示计算进度
- **号码走势图**：可视化展示每期开奖号码分布、遗漏、频次统计
- **历史记录**：完整保存每期开奖数据，含中奖等级、中奖注数、投入成本、回本率
- **数据持久化**：所有数据通过 localStorage 持久化存储，刷新不丢失

## 技术栈

| 技术 | 用途 |
|------|------|
| React 18 | UI 框架 |
| Vite | 构建工具 |
| Web Worker | 多线程并行计算 |
| localStorage | 数据持久化 |
| CSS3 | 动画与样式 |

## 项目结构

```
src/
├── components/
│   ├── BallSelector.jsx    # 号码选择组件
│   ├── DrawPanel.jsx       # 开奖面板（进度条、结果展示）
│   ├── HistoryList.jsx     # 历史记录与财务统计
│   └── TrendChart.jsx      # 号码走势图
├── hooks/
│   └── useLottery.js       # 核心业务逻辑（多 Worker 管理、结果合并）
├── utils/
│   └── lottery.js          # 双色球规则定义与工具函数
├── workers/
│   └── drawWorker.js       # 后台计算 Worker（分块处理）
├── App.jsx
└── main.jsx
```

## 中奖规则

| 奖级 | 红球 | 蓝球 | 奖金 |
|------|------|------|------|
| 一等奖 | 6 | 1 | 500 万元 |
| 二等奖 | 6 | 0 | 20 万元 |
| 三等奖 | 5 | 1 | 3,000 元 |
| 四等奖 | 5 / 4 | 0 / 1 | 200 元 |
| 五等奖 | 4 / 3 | 0 / 1 | 10 元 |
| 六等奖 | 2/1/0 | 1 | 5 元 |
